#############
Configuration
#############

Happyschool disposent de trois différents niveaux de configuration :

-  Le plus bas niveau, dans ``happyschool/settings.py``, est la
   configuration liée à django et se rapporte à la base de donnée, les
   applications utilisées, le serveur de courriel sortant, etc.
-  Le niveau intermédiare est simplement des entrées dans la base de donnée.
   Par exemple, les types de sanctions ou les motifs des appels.
-  Finalement, le plus haut niveau est accessible directement depuis l’interface web.
